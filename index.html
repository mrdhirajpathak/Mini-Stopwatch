How to operate

Quick controls and keyboard shortcuts to run the stopwatch fast.

  • Start → Begin time counting.
  • Stop → Pause the timer.
  • Lap → Record current time in lap list.
  • Reset → Clear all time and lap data.

Keyboard shortcuts

  • Space = Start/Stop
  • L = Lap
  • R = Reset
