A lightweight and responsive Mini Stopwatch Web Application built using HTML, CSS, and JavaScript.
This project was developed as a college mini project to demonstrate the working of time-based JavaScript functions and DOM manipulation.

The stopwatch allows users to start, pause, and reset time tracking, making it a simple yet practical example of interactive frontend development.

---

🚀 Features

✔ Start the stopwatch
✔ Pause the stopwatch
✔ Reset the timer
✔ Accurate time tracking
✔ Clean and minimal user interface
✔ Fast and lightweight application

---

🛠 Tech Stack

Frontend Technologies

- HTML5 – Structure of the application
- CSS3 – Styling and layout design
- JavaScript – Stopwatch logic and timer functionality
