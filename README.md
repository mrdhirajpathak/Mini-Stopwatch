A lightweight and responsive Mini Stopwatch Web Application built using HTML, CSS and JavaScript.
This project was developed as a college mini project to demonstrate the working of JavaScript timer functions and DOM manipulation.

The stopwatch allows users to start, pause and reset time tracking with a simple and clean interface.

---

🚀 Features

- Start the stopwatch
- Pause the stopwatch
- Reset the timer
- Accurate time tracking
- Simple and user friendly UI

---

🛠 Tech Stack

Frontend Technologies:

- HTML5
- CSS3
- JavaScript (Vanilla JS)

⚙️ How to Run the Project

Clone the repository

git clone https://github.com/mrdhirajpathak/Mini-Stopwatch.git

Open the project folder

cd Mini-Stopwatch

Run the project by opening index.html in your browser.
