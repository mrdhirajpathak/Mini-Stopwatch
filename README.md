# ⏱ Mini Stopwatch

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Project](https://img.shields.io/badge/Project-College%20Mini%20Project-green)

A lightweight and responsive **Mini Stopwatch Web Application** built using **HTML, CSS and JavaScript**.

This project was developed as a **college mini project** to demonstrate how JavaScript timer functions work and how DOM manipulation can be used to build interactive web applications.

---

## 🚀 Features

- Start the stopwatch  
- Pause the stopwatch  
- Reset the timer  
- Accurate time tracking  
- Clean and simple user interface  

---

## 🛠 Tech Stack

Frontend Technologies:

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## ⚙️ How to Run the Project

### Clone the repository

```bash
git clone https://github.com/mrdhirajpathak/Mini-Stopwatch.git
```

### Go to the project directory

```bash
cd Mini-Stopwatch
```

### Run the project

Open **index.html** in your web browser.
